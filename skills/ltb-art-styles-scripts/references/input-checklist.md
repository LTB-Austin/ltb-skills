# Art Styles & Scripts — What to Give Claude (Austin's cheat sheet)

One message, no image prep needed. Claude builds the full deck with labeled placeholder frames; you drop the artwork in afterward in PowerPoint.

## 1. Style notes (in the message or a doc)
Per art style:
- Style name (or ask Claude to propose one)
- The visual idea in rough bullets — world, palette, how the MOA plays out
- What images you plan to drop in (so frame captions are right), e.g. "heartburn visual, isolated stomach, closeup biology, science moment, product showcase"
- MOA part count if you know it

## 2. Scripts doc
Any format. Label each script with name + length ("Ad – 15 seconds"). Placed verbatim, including alternates and asterisked disclaimers.

## 3. Brand Archive zip
Standard LTB logos + mitosis zip. Include the client logo if you have it; otherwise it gets a placeholder frame too.

That's it — Claude writes the Briefs, MOA narration, Style Details, and all frame captions from your notes, using the measured Zegerid layout spec.
