# LTB Brand System (current — 2026 rebrand)

Use this for every client-facing LTB deck. It supersedes the older palette (the retired teal `#008575` and its siblings). If you see those old hexes in an input, do not reuse them.

## Surfaces & type
- Slide background: `#f0f0f0` (never pure white).
- Text/ink: `#0C0D10` (effectively black). Body copy is always ink on `#f0f0f0`.
- Font: **Lexend** throughout. Headings 600/700 weight; body 300 (light); labels/eyebrows 500, uppercase, letter-spaced.
- No emojis, ever. No corner chevrons.
- Divider: 3pt solid black rule, running ~2/3 across the slide.

## The mitosis gradient (signature)
`linear-gradient(110deg, #94CAB4 0%, #7FB6CC 18%, #7C82A1 38%, #8C6C8F 52%, #E1483B 70%, #EE6B4D 86%, #F2966A 100%)`

Reserve the gradient for emphasis only: a single key word, a hero statistic, a thin accent rule, or a card border. Never run body copy or whole headlines in gradient. On a light deck it reads as the brand's "cell → division" energy; keep it rare so it stays special.

## Imagery
- Mitosis / dividing-cell motifs are the house visual language (the brand is "science multiplying"). Use the brand cell renders/photography where available; otherwise a soft gradient blob motif.
- Study/data visuals stay clean and editorial — lots of `#f0f0f0` breathing room, one idea per slide.

## Voice
Confident, science-forward, plain. "Science leads, creativity amplifies." Specific over vague; never hype. Let the evidence carry the weight.

## Card / layout conventions
- Cards: `#f0f0f0` fill, thin ink or gradient border, generous padding, rounded corners.
- Eyebrow (small uppercase label) → headline → body → source line, top to bottom.
- One-idea-per-slide. Whitespace is a feature, not a gap to fill.
